# goTestRedis

## Usage
```
> ./bin/tredis --help
Usage of ./bin/tredis:
  -addr string
    	The address of Redis (default "127.0.0.1")
  -db int
    	use DB, Default is 0
  -key string
    	The name of key. Default is foo (default "name")
  -pool int
    	The poolsize,if poolsize is 0,don't use pool (default 50)
  -port int
    	The port number of Redis (default 6379)
  -threads int
    	The number of threads. (default 200)
```
